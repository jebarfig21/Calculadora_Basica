Este programa esta escrito en java, el paquete se llama Calculadora_basica, por lo que la forma de compilar es desde la carpeta padre de Calculadora basica con el comando "javac Calculadora_Basica/*.java", mientras que la ejecución, seria de la siguiente forma, "java Calculadora_Basica.Main".

A si mismo, este programa es de uso libre y abierto a todo público, se pueden usar fragmentos de código siempre y cuando se escriba la colaboracion de mi código, agradeceria feedbacks.

A su vez tome ejemplos para iniciar el JFrame del siguiente link: https://gist.github.com/alvareztech/9022849

El fin de este programa fue continuar con la práctica de este lenguaje asi como recordar varios términos básicos.
